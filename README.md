# Travel Website

## Project Description
This project is a travel website designed to offer a variety of amazing destinations for different types of travelers. The website features user-friendly navigation, a creative design, and an engaging layout that aims to attract visitors and encourage them to explore more.

## Pages Overview

### 1. **Home Page**
The **Home Page** serves as a welcoming gateway to the website, featuring:
- A navigation menu for easy access to other pages.
- A display of popular tours to entice users to explore.
- A video ad promoting the companyâ€™s services.
- A sneak peek of recent blog posts.
- A footer for additional information and links.

### 2. **About Us Page**
The **About Us Page** introduces the company, showcasing:
- A brief paragraph describing the services offered.
- The company's mission and vision, providing insight into its values and goals.
- A navigation menu for smooth transitions to other pages.

### 3. **Tour Page**
The **Tours Page** enables users to:
- Search for tours using filters based on price range, continent, and number of days.
- Explore a world map where users can click on specific regions to discover available tours.
- Utilize a selection functionality that adds chosen tours to a table on the **Book Page**, helping users keep track of their selections.

### 4. **Blog Page**
The **Blog Page** features:
- A navigation menu for seamless navigation.
- An authentication feature for users to submit their own travel experiences.
- An interactive platform for users to share their adventures and insights.

### 5. **Book Page**
The **Book Page** allows users to:
- Book their dream destinations using an easy-to-fill form.
- View a table listing the selected tours, which were previously explored on the **Tours Page**.
- This combination of a table and form helps users compare their options before confirming a booking.

## Project Requirements
To run and develop the website, make sure you have the following:

1. **Visual Studio Code**: An IDE for editing your website's files.
2. **Live Server Extension**: Install the "Live Server" extension in Visual Studio Code to preview changes in real-time.
3. **Project Setup**: Open the project folder in Visual Studio Code to ensure all functionalities of the website perform well.

## Getting Started

1. Clone or download this repository to your local machine.
2. Open the project folder in **Visual Studio Code**.
3. Install the **Live Server** extension from the VS Code marketplace.
4. Launch the project using **Live Server** to see the website in action.
